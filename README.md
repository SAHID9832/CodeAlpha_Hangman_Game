## Project Overview
This project is a simple text-based Hangman game developed using Python as part of the CodeAlpha Python Programming Internship. The player guesses a hidden word by entering one letter at a time.

## Game Rules
- A word is randomly selected from a predefined list of 5 words  
- The player has **6 incorrect attempts**  
- Repeated letter guesses are detected and **do not reduce chances**  
- Only single alphabet letters are accepted  

## Technologies Used
- Python
- random module
- time module

## Concepts Used
- Lists and strings  
- While loop  
- If-else conditions  
- User input validation  
- Basic game logic
